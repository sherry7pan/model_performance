This folder include the following files:

1. data_analysis.ipynb
   A jupyter notebook containing a thorough  data analysis and visualizations. 

2. model.py
   Process for elevating the initial model with cross validated score.

3. grid_search_model.py
   Improved model by tuning the hyper parameters of the random forest classifier      using grid search.

4. prediction.py
   This program returns the predicted values along with features as a cvs file.
 
5. random_forest_classifier.pkl
   The final tuned classifier saved as a pickle file 

6.prediction.csv
   cvs file for use to evaluate the true AUC with test dataset.
